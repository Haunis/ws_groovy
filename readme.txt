安装:groovy
  1.安装 SDKMAN! 
      curl -s "https://get.sdkman.io" | bash
  2.新开一个终端或者在当前终端中输入以下指令
      source "$HOME/.sdkman/bin/sdkman-init.sh"
  3.安装当前最新稳定的groovy版本
      sdk install groovy
  4.安装完成后运行查看版本信息(如：Groovy Version: 2.4.12 JVM: 1.8.....)
      groovy -version

运行groovy文件:
  groovy xxx.groovy
